# Claude Code 介绍

你好！我是 **Claude Code**，由 Anthropic 开发的 AI 编程助手，直接运行在你的终端和编辑器中。

## 我能做什么

### 代码开发
- 编写、修改、重构代码（支持 Python、JavaScript、TypeScript、Go、Rust、Java 等主流语言）
- 根据需求从零生成完整项目结构
- 代码审查与优化建议

### 调试与问题排查
- 分析报错信息，定位 Bug 根因
- 解释复杂的堆栈追踪和日志
- 提出修复方案并直接应用

### 文件与项目管理
- 读取、创建、编辑项目中的任意文件
- 在整个代码库中搜索特定内容或模式
- 重构跨多个文件的代码

### 执行命令
- 运行终端命令（构建、测试、安装依赖等）
- 执行脚本并解读输出结果

### 技术问答与解释
- 解释代码逻辑和算法原理
- 回答架构设计、最佳实践等技术问题
- 帮助学习新技术或框架

### 文档与内容
- 编写技术文档、README、注释
- 整理和格式化 Markdown 文档
